La ruta POST /reporte recibe datos JSON en el cuerpo de la solicitud y devuelve una respuesta JSON con el estado del reporte y el mensaje recibido.
Esta ruta puede servir en una aplicación comunitaria para enviar reportes, alertas o información desde los vecinos y que el servidor los reciba de forma organizada.
Por ejemplo, un ciudadano puede enviar un reporte de un problema local y la API lo guarda o lo envía a un sistema de gestión de la comunidad.
